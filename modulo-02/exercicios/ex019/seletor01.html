Seletores Personalizados

Para criar estilos personalizados, precisamos aprender a utilizar os seletores de id (#) e de class (.) de maneira eficiente.

Selector id (#)

Com o seletor id identificamos um determinado elemento único, criando um estilo especial para esse elemento isolado.

<h1 id="principal">Criando Sites em HTML e CSS</h1>

Selector class (.)

Já com um class nós agrupamos elementos múltiplos que tenham caracteríticas semelhantes, uma classe vai identificar à qual um ou mais elementos perteçam e assim compartilhando características em comum a todos os os que faça parte deese grupo.

HTML básico - <h2 class="basico">HTML</h2>

HTML intermediário - <h2 class="intermediario">HTML</h2>

HTML avançado - <h2 class="avancado">HTML</h2>

CSS básico - <h2 class="basico">CSS</h2>

CSS intermediário - <h2 class="intermediario">CSS</h2>

CSS avançado - <h2 class="avancado">CSS</h2>

Se liga na regra:

Em um mesmo documento HTML, só podemos usar um id para um único elemento. Porém, podemos atribuir um mesmo class para vários elementos que possuam essa mesma característica.

Mais uma regra:

Todo id em HTML e identificado na CSS por um símbolo #.

Toda class em HTML é identificada nas CSS por um ponto.
