Fim do exemplo

Para esconder o parágrafo que está dentro da div você deve usar esse parâmetro >, que significa que o p é filho de uma div.


        div > p{
            display: none;
        }

Para que o parágrafo <p> que está dentro da <div> apareça, deve-se colocar uma pseudo-classe, mais o parâmetro > que indica de o <p> é filho da <div>.


        div:hover > p{
            display: block;
            color: white;
            background-color: red;
            font-size: 3em;
            width: 510px;
        }
    

        div:hover{
            color: red;
        }
