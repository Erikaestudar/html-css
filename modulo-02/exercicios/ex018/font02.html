Adicionando fontes externas

Caso não tenha achado sua fonte no Google Fonts você poderá baixar no site dafont.com.

@font-face {
    font-family:"Old London Font" ;
    src: url("font/OldLondon.ttf") format("truetype"), 
        url("font/Olondon_.otf") format("opentype");
    font-weight: normal;
    font-style: normal;
        }        

    Tipos de format()
    
  • opentype (otf) existe fontes opentype que são ttf
    
  • truetype (ttf)
    
  • embedded-opentype
    
  • truetype-aat (Apple Advanced Typography)
    
  • svg

Para usar nossa própria fonte utilizamos um seletor especial/ regra chamado @font-face{} na propriedade font-family: "nome"; você pode estar colocando o nome que preferir, lembrando de repetir o mesmo nome em outro propriedade.
