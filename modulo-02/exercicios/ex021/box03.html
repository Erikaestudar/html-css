Teste de borda Repeat


        h1{
            border: 20px solid transparent;
            padding: 10px;
            border-image-source: url("borda.png");
            border-image-slice: 26;
            border-image-repeat: repeat;

            /*Shorthand*/
            border-image: url("borda.png") 26 repeat;
        }

É necessário ter esse código para segurar a borda border: 20px solid transparent; se tirá-la ela deixa de existir. Pode ocorrer de alguns navegadores não ser compatível se colocar cor e imagem na borda, podendo ser uma boa opção deixar o border transparent.
