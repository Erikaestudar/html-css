Faça uma escolha

Iremos montar uma página com menus para abrir no iframe, anteriormente você aprendeu a abrir outras páginas em uma aba separada usando o target="_blank".

  • Primeira página
  • Segunda página
  • Terceira página

Exemplo: 

    <ul>
        <li><a href="paginas-extras/pg001.html" target="frame">Primeira página</a></li>
        <li><a href="paginas-extras/pg002.html" target="frame">Segunda página</a></li>
        <li><a href="paginas-extras/pg003.html" target="frame">Terceira página</a></li>
    </ul>

Montamos um menu assim, nota que colocamos no lugar do valor _blank o valor frame. Que é o valor do atributo name do <iframe> 

        <iframe id="tela" name="frame" src="" frameborder="0"></iframe>
