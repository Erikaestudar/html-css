Imagem dinâmica

Para adaptar nosso conteúdo ao tamanho da tela, devemos gerar imagens de tamanho diferentes e fazer o navegador carregar a imagem certa para cada situação. Para isso, usaremos as tags <picture> e <source>.

Para esse exemplo, criamos as três imagens abaixo: a menor tem 300x300px, a média tem 700x700px e a maior tem 1000x1000px. Com o seguinte código base:

  
    <picture>
        <img scr="foto-g.png" alt="A imagem flexível funciona">
    </picture>

Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador.

O problema vai começar a surgir quando a janela do navegador chegar perto dos 1000 pixels de largura, pois a foto não vai mais caber lá. Para resolver esse problema vamos adicionar uma linha:

NOTE: Que mudamos o min-width pelo max-width.
<picture>
    <source media="(max-width:1050px)" srcset="foto-m.png" type="image/png">
    <img src="foto-g.png" alt="A imagem flexível funciona">
</picture>

A tag <source> possui três atributos:

  • media indica o tamanho máximo a ser considerado para carregar a imagem indicada no atributo srcset.
  • srcset vai configurar o nome da imagem que será carregada quando o tamanho indicado for atingido
  • type vai indicar o media type da imagem que usamos (veja mais informções sobre media types no capítulo 10 ou no ex010)

Atenção!

você pode até colocar o valor exato de 1000px na propriedade max-width, mas vai perceber que um valor ligeiramente acima vai gerar resultados mais interessantes.

Você vai perceber que a imagem muda automaticamente conforme aumentamos ou diminuímos o tamanho da tela.
Vamos continuar e acrescentar mais um <source> à nossa imagem:

NOTE: Que mudamos o min-width pelo max-width.
<picture>
    <source media="(max-width:750px)" srcset="foto-p.png" type="image/png">
    <source media="(max-width:1050px)" srcset="foto-m.png" type="image/png">
    <img src="foto-g.png" alt="A imagem flexível funciona">
</picture>

É importante que exista uma ordem entre os <source>, e nessa configuração, os itens mais acima sejam os menores tamanhos para o max-width e os seguintes sejam maiores, de forma crescente. O último item dentro de <picture> deve ser a imagem padrão.
