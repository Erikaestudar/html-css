Trabalhando com Listas

Listas Ordenadas

Ordered List <ol> (essa tag sempre deixará sua lista ordenada, não importando se mais tarde você tiver que acrescentar mais itens) and List Item <li>.

  1. Acordar
  2. Ligar para o João
  3. Escovar os dentes
  4. Tomar café

A tag <ol> possui um parâmetro type, onde configuramos o tipo de marcador da lista atual. As opções de valores para esse parâmetro são:

  • 1 - Valor padrão. Cria listas numeradas. Ex:1, 2, 3, 4, ...
  • A - Cria listas alfabéticas em maiúsculas. Ex: A, B, C, D, ...
  • a - Cria listas alfabéticas em minúsculas. Ex: a, b, c, d, ...
  • I - Cria listas com algarismos romanos em maiúsculas. Ex: I, II, III, IV, ...
  • i - Cria listas com algarismos romanos em minúculas. Ex: i, ii, iii, iv, ...

Você também pode indicar o início da contagem usando o parâmetro start.

Ex:

  1. Lavar roupa
  2. Almoçar
  3. Estudar programação

Listas não Ordenadas

Unordered List ul (são aquelas onde a ordem dos itens não influenciará no significado da lista)

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

O marcador padrão é bolinha preta (disc), mas existe a opção de configurar a propriedade type da tag <ul> com os seguintes valores:

  • disc - padrão. Uma bola preta totalmente pintada
  • circle - Uma bola com uma borda preta e sem preenchimento
  • square - Um pequeno quadrado preto totalmente pintado
  • none - Tira os marcadores da lista
