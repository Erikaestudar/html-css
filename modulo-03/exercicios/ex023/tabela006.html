Tabelas em telas pequenas

Envelopa a tag table com uma div e coloca um id, fazendo assim só a tabela vai ter rolagem lateral se tornando responsiva em telas pequenas. 

  
    div#container{
        overflow-x: auto;
    }

Agrupando Colunas com <colgroup>, cada coluna poderá ter uma cor diferente.


    col.cnome{
        background-color: rgba(25, 179, 89, 0.527);
    }

    col.cgrupo{
        background-color: rgba(127, 255, 212, 0.712);
    }

    col.cprof{
        background-color: rgba(0, 128, 0, 0.582);
    }


    <colgroup>
        <col class="cnome">
        <col class="cgrupo" span="2">
        <col class="cprof">
    </colgroup>
