    Anatomia para tabelas grandes

    <caption> É a legenda da tabela<;caption>

    <table>
        <thead>
            tr,td,th
        <tbody>
            tr,td,th
        <tfoot>
            tr,td,th
    </table>

    O scope é muito importante para que os mecanismos de busca consigam analisar os dados da tabela de forma coerente.
    O <th scope="col"> simboliza que o Estado é título da coluna.

    <thead>
        <tr>
            <th scope="col">Estado</th>
            <th scope="col">População</th>
        </tr>
    </thead>


    O <th scope="row"> simboliza que o Total de Habitantes é título da linha.

    <tfoot>
        <tr>
            <th scope="row">Total de Habitantes</th>
            <td class="num">210.147.125</td>
        </tr>
    </tfoot>

    Efeito Zebrado - usasse uma pseudo-classe :nth-child(..n)
    
    tbody < tr:nth-child(2n){
        background-color: lightgray;
    }

    No exemplo acima está de duas em duas, mas pode colocar de três, cinco e etc.

    Para números ímpares.
    tbody < tr:nth-child(odd){   ou  /* (2n-1) */
        background-color: lightgray;
    }

    Para números pares.
    tbody < tr:nth-child(even){  ou /* (2n) */
        background-color: lightgray;
    }
